# FinTrack

Simple personal finance web app (MVP) built with Python standard library + SQLite.

## Features
- Email/password auth
- Transaction CRUD
- Monthly dashboard (income, expense, net)
- Monthly budget tracking
- Category breakdown
- Date/category filters
- CSV export

## Run
1. Ensure Python 3.10+ is installed.
2. From project root:

```bash
python app.py
```

3. Open `http://localhost:8000`.

## Notes
- Data is stored in `data/fintrack.db`.
- This is an MVP for local/self-hosted use.
